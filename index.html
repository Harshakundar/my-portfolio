Selected Work

Projects I've shipped

/01

LAN Print Bridge

A resilient printing system connecting a Laravel backend to physical printers on counter PCs. A C# client app on each counter listens over the LAN, with Laravel Reverb broadcasting print jobs over WebSockets. The whole system keeps working when the internet drops, as long as the local network is up — backed by a static-IP Ubuntu server running Nginx and Supervisor, and self-hosted fonts/CSS so PDF generation never waits on a CDN.

C#LaravelReverb / WebSocketsNginxSupervisorUbuntu

Photoshop Batch-Export Automation

An automation pipeline for a fashion e-commerce workflow. Two ExtendScript batch-export scripts run inside Photoshop, paired with a FastAPI service that classifies product images using a YOLO machine-learning model — turning a tedious manual export process into a hands-off pipeline.

PythonFastAPIExtendScriptYOLO / MLpywin32

Flutter Blog App

A full-featured blog platform targeting Android, iOS, and Web built with Flutter and Feature-First Clean Architecture. Supabase handles auth, database, and file storage; BLoC/Cubit manages state across all layers; Hive caches posts locally so the app remains fully readable without a network connection.

FlutterDartSupabaseBLoC / CubitHiveClean Architecture
