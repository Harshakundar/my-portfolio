Selected Work

/01

LAN Print Bridge

A resilient printing system connecting a Laravel backend to physical printers on counter PCs. A C# client app on each counter listens over the LAN, with Laravel Reverb broadcasting print jobs over WebSockets. The whole system keeps working when the internet drops, as long as the local network is up — backed by a static-IP Ubuntu server running Nginx and Supervisor, and self-hosted fonts/CSS so PDF generation never waits on a CDN.
